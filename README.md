## Overview
Base de datos estructurada sobre el uso de inteligencia artificial generativa (IAG) en investigación académica y docencia universitaria en América Latina (2022-2025). Incluye datos sobre: frecuencia de uso por estudiantes e investigadores, áreas de aplicación (redacción, análisis de datos, programación), percepción de fiabilidad y ética, e impacto en productividad científica.
